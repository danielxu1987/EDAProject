Mart down provides an easy way to make standard types of formatted text, like

  • italics
  • bold
  • code
  • links
  • image

But did you know that you can also use R Markdowns markdown to make

  • Latex equations, \(E = mc^{2}\)
  • And bibliographies (R Core Team 2019)

R Markdown

This is an R Markdown document. Markdown is a simple formatting syntax for authoring HTML, PDF, and MS Word documents. For more details on using R Markdown see http://rmarkdown.rstudio.com.
